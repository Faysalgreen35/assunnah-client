# 🚀 As-Sunnah App - Quick Start Guide

## Install in 4 Steps

### 📱 Android
1. Open **Chrome** → Visit **assunnah.com**
2. Tap **⋮** (menu) → **"Install app"**
3. Tap **"Install"**
4. ✅ Done!

### 🍎 iPhone  
1. Open **Safari** → Visit **assunnah.com**
2. Tap **⬆️** (Share) → **"Add to Home Screen"**
3. Tap **"Add"**
4. ✅ Done!

### 💻 Desktop
1. Open your browser → Visit **assunnah.com**
2. Click **Install** icon in address bar (or menu)
3. Click **"Install"**
4. ✅ Done!

## Troubleshooting

| Problem | Fix |
|---------|-----|
| Won't install | Update browser, clear cache, try WiFi |
| Loads slowly | Close other apps, restart phone |
| Can't see new products | Force close app, reopen, clear cache |
| Lost cart/wishlist | Sign in to your account — data reappears |
