⚙️ SETUP STEPS (IN ORDER)
═══════════════════════════════════════════════════════════════════════════

PART 1: LOCAL SETUP (Your Computer)

1. Install Git from git-scm.com
   Verify: git --version

2. Configure Git
   git config --global user.name "Your Name"
   git config --global user.email "your@email.com"

3. Create SSH Key
   ssh-keygen -t ed25519 -C "your@email.com"
   Press Enter twice (no passphrase)

4. Copy Public Key
   Windows: Get-Content ~/.ssh/id_ed25519.pub | Set-Clipboard
   Mac/Linux: cat ~/.ssh/id_ed25519.pub

5. Add Key to GitHub
   github.com → Settings → SSH keys
   Paste your public key → Save

6. Test SSH Connection
   ssh -T git@github.com
   Should see: "Hi YourName! You've successfully authenticated"


PART 2: INITIALIZE PROJECT (Your Folder)

7. Initialize Git in Project
   cd F:/NextZen/zico/assunnah-client
   git init

8. Make First Commit
   git add .
   git commit -m "Initial commit: Add Next.js app with PWA"

9. Create GitHub Repository
   Go to github.com → + icon → New repository
   Name: assunnah-client
   Click "Create repository"

10. Connect to GitHub
    git remote add origin git@github.com:YOUR-USERNAME/assunnah-client.git
    git branch -M main
    git push -u origin main


PART 3: VERCEL DEPLOYMENT (Cloud)

11. Connect Vercel to GitHub
    Go to vercel.com → Add New → Project
    Select your assunnah-client repository
    Click "Deploy"
    Wait 2-5 minutes for build

12. Get Your Live URL
    https://assunnah-client-xyz.vercel.app
    (Or add custom domain later)
